# Tickety
## What is Tickety?
Tickety is an ECommerce website that sells business customers a solution to displaying products. For example a 
bakery display. 

##Technologies
1. React
2. Bootstrap
3. NodeJS
4. mySQL
5. Express

## Upcoming Features
